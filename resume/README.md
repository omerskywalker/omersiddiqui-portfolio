# Omer Siddiqui — Interactive Resume

  A beautiful, accessible interactive resume web app built with React + Vite + Tailwind CSS.

  ## Features
  - Animated hero with stats cards
  - Expandable experience timeline
  - Skills pills with category grouping
  - Personal projects section with tech stacks
  - "How I Work" philosophy section
  - Dark mode toggle
  - ATS-friendly PDF download (tagged, screen-reader accessible)
  - Fully responsive (mobile-first)

  ## Tech Stack
  - React 18 + TypeScript + Vite
  - Tailwind CSS + shadcn/ui
  - `@react-pdf/renderer` for PDF generation
  - `lucide-react` + `react-icons` for icons

  ## Getting Started
  ```bash
  npm install
  npm run dev
  ```

  ## Deploy
  ```bash
  npm run build   # outputs to dist/
  ```
